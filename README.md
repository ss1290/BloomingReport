# Steps to Run the Project

1) Open terminal and run "npm i" command to install all npm modules
2) Run "npm start" to launch the app in your device.

# App Flow Description

The app flow for adding a Blooming Report consists of **five main steps**:

1) **Farmer Profile**  
2) **Land Details**  
3) **Crop Details**   
4) **Flowering Details**.  
5) **Beekeeping Scope**

After completing the 5th step, the form is saved successfully.  
All saved reports can be viewed anytime under the **"Saved Reports"** bottom tab.
